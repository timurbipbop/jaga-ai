# HEARTBEAT.md — SoulMD Daily Checks

## Morning Check (8AM WIB)
1. Send gentle good morning health reminder to parents (via WhatsApp when linked)
2. Remind about morning medications if any
3. Suggest morning walk / light movement

## Evening Check (8PM WIB)
1. Evening medication reminder
2. Check in on how the day went (symptoms, mood, food)
3. Log any reported vitals

## Proactive Monitoring Tasks (rotate)
- Review recent memory files for any concerning trends
- Update patient profiles with new information
- Prepare weekly health summary for Timmy
- Research relevant health topics (elderly care, autoimmune management, fall prevention)

## Always Watch For
- Any mention of chest pain, breathing difficulty, confusion, fall → ESCALATE IMMEDIATELY to Timmy
- Missed medication doses → gentle reminder + log
- Changes in appetite or sleep → log and note trend
- Emotional withdrawal or sadness → respond with warmth, note pattern
